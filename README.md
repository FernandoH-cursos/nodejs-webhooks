# Webhooks con Nodejs

Backend que consume _webhook_ de github para que podamos recibir mensajes dependiendo del evento de
github que desencadenemos. Además de enviar estos mensajes a un _webhook_ de discord y que se muestren en un canal.

## Temas a tratar:

- #### ¿Que son los webhooks?
- #### Configuración de Webhooks de _Github_ y _Discord_.
- #### Recibir mensajes a nuestro server de discord.
- #### Envío de imagenes a discord.
- #### Headers de petición personalizados para webhook.
- #### Middleware de autenticación para webhook.
- #### Bloqueo de peticiones no válidas.
